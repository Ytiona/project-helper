# Project Helper
项目助手

### 目前规划功能：
  1.自动将psd设计图构建出前端项目的基础html、css（vue、react、小程序），集成了tinypng的压缩功能；
  2.tinypng压缩，已破解限制；
  3.psd文件解析，自动将psd文件中的图层逐一导出为png图片；

### 目前已完成内容：
  1.基础ui组件的封装；
  2.psd文件的选中交互；
  3.psd文件的解析；
  4.构建配置的部分交互；


## 技术栈：vue3、psdjs、vite、nodejs、electron、iconfont、lodash、less、vue-router
